在 ArchLinux 中使用 Clash

Posted Sep 6, 2021

达到的效果如下

安装

  • 安装 Clash

      paru -S clash-dev-git clash-dashboard

  • 配置 Clash 订阅

    • 从 订阅更新 Clash 节点

            wget -O ~/.config/clash/config.yaml someurl.com

    • 编辑 clashup.json

          {
      "http_port": 7890,
      "socks5_port": 7891,
      "redir_port": 7892,
      "allow_lan": true,
      "external_controller": "127.0.0.1:9090",
      "secret": 1234,
      "subscribe_url": "yoururl",
      "is_subscribe_banned": false,
      "custom_rules": [],
      "mmdb_file_url": "http://www.ideame.top/mmdb/Country.mmdb",
      "mmdb_version_url": "http://www.ideame.top/mmdb/version"
    }

    • 启动 Clash

          cd ~/.config/clash

    clash -d .

    • 在终端中使用代理

          export https_proxy=http://127.0.0.1:7890
    export http_proxy=http://127.0.0.1:7890

    • 设置浏览器代理
